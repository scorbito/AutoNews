🔑 앱 비밀번호(IMAP) 발급 방법 — 처음이면 꼭 읽어주세요
로그인 비밀번호가 아니라 메일 앱 비밀번호가 필요합니다. 그리고 메일에서 IMAP 사용을 켜야 해요.

네이버

① 메일 → 환경설정 → POP3/IMAP 설정 → IMAP/SMTP 사용 ON
② (2단계인증 사용 시) 네이버 내정보 → 보안 → 애플리케이션 비밀번호에서 발급한 16자리를 비밀번호 칸에 입력
설정: mail.naver.com → 환경설정 / 보안: nid.naver.com

다음/카카오

① 다음메일 → 환경설정 → IMAP/POP3 → 사용
② (카카오 2단계인증 시) 카카오계정 보안에서 앱 비밀번호 발급해 입력

Gmail

① Google 계정 2단계 인증 켜기 → 앱 비밀번호 생성(myaccount.google.com/apppasswords)
② 생성된 16자리를 비밀번호 칸에 입력 (IMAP은 Gmail 설정에서 사용 ON)

※ 어려우면 관리자에게 요청하세요 — 대신 설정해 드립니다.
